⛊IMESupport

解决输入法不跟随

Installation

  • 从github上下载IMESupport-master.zip
  • 将IMESupport-master.zip解压，复制到C:\Users\chq_p\AppData\Roaming\Sublime Text 3\Packages目录下 (即Perference > Browse Packages 打开的位置)。

⛊SublimeHighlight

高亮复制sublime中的代码到word中

Ho to do

首先打开包管理器：CTRL+SHIFT+P

输入add

添加下面url：
https://github.com/n1k0/SublimeHighlight/tree/python3

然后再次按下CTRL+SHIFT+P，输入install

查找SublimeHighlight按下回车安装

选中需要拷贝的文字，右键选择复制为HTML，就可以在word中粘贴了

复制格式如何设定

点击菜单栏中首选项（preference）->Package Settings->SublimeHighlight->Settings-User，将以下代码复制到打开文件中并保存：

{
“theme”: “monokai”,
“linenos”: “inline”,
“noclasses”: true,
“fontface”: “Menlo”
}

注

  • 本质上这个插件，是以RTF或者HTML的形式复制
  • 除了用sublimetext 还可以用这个网址http://www.planetb.ca/syntax-highlight-word

⛊Markdown HTML Preview

ctrl+shift+p，点击Package Control: Install Package,输入Markdown HTML Preview。
在sublime中打开任意一个markdown文档，按ctrl+shift+m,即可转换成HTML文档。
