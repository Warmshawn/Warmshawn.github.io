配置Git、SSH

  • 下载、安装 Git

  • 绑定用户

  • 配置 SSH

    • 第一次使用，按照上述操作获得SSH Key。Git会提示如何操作，密码可以空。SSH Key保存在C:\Users\Administrator\.ssh。推荐将.ssh保存至一个专门的地方，下次使用时copy至C:\Users\Administrator即可。

    • .ssh 目录，里面有 id_rsa 和 id_rsa.pub 两个文件，这两个就是SSH Key 的秘钥对，id_rsa 是私钥，不能泄露出去，id_rsa.pub是公钥，可以放心地告诉任何人。用记事本打开 id_rsa.pub，得到ssh key 公钥。

    • 为 Github 账户添加 ssh key 。登录 Github，展开个人头像的小三角，点settings，然后打开SSH keys菜单，点击Add SSH key新增密钥，填上标题。

  • 验证账户
